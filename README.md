# Project name E-Bazar

> Short description of the project.

- Login, Register (for Admin and Customer)

* Dashboard for Admin

- Add Products, View Products, Delete Products for Admin

* Cart, Checkout, Order for Customer

- Manage User, Delete User

## Built With

- React JS
- React Router
- React Hook Form
- React Icons
- Axios
- Tanstack Query
- Tailwind CSS
- Daisy UI
- Firebase
- Sweetalert2
- Tanstack Query
- React-Rating
